# PetShop 🐾

A React Native CLI application built to manage pet shop services and products.  
This app demonstrates cross-platform mobile development using React Native.

---

## 🚀 Features
- Browse available pets and products
- Add items to cart
- Manage orders
- Cross-platform support (iOS & Android)

---

## 📦 Installation

### Prerequisites
- Node.js (>= 18)
- React Native CLI
- Android Studio (for Android builds)
- Xcode (for iOS builds)
- Git
